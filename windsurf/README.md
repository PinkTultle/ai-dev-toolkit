# Windsurf 지침

> 아직 지침 파일이 작성되지 않은 placeholder 디렉토리입니다.

## 설정 형식

- **글로벌**: `~/.windsurf/rules/*.md`
- **프로젝트 (권장)**: `<project>/.windsurf/rules/*.md` (다중 파일)
- **프로젝트 (대안)**: `<project>/.windsurfrules` (단일 파일)
- 파일당 12,000자 제한

## 배포 위치

| 소스 | 배포 위치 |
|------|-----------|
| `global_rules.md` | `~/.windsurf/rules/global_rules.md` |
| `*.md` | `<project>/.windsurf/rules/` |
